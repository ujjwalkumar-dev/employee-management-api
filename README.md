# Employee Management System (ASP.NET Core)

## Overview
A backend system built using ASP.NET Core Web API following Clean Architecture principles.
Designed to demonstrate scalable, secure, and maintainable backend development.

## Features
- JWT-based authentication and authorization
- Role-based access control
- Employee CRUD with validation
- Global exception handling and logging

## Tech Stack
- ASP.NET Core Web API
- C# / ADO.NET
- SQL Server
- JWT Authentication

## Architecture
- Domain: Business entities
- Application: Interfaces and business logic
- Infrastructure: Data access and logging
- API: Controllers and middleware

## How to Run
1. Clone repository
2. Update connection string
3. Run SQL scripts
4. Start the API
